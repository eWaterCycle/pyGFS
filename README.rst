pyGFS
=====

What is pyGFS:
~~~~~~~~~~~~~~

pyGFS is a python library to download weather forecasts from the Global
Forecast System (GFS).

Installation
~~~~~~~~~~~~

pyGFS is installable via pip:

::

   pip install pygfs
